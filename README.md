Descripción del proyecto.

**Atlántli es una plataforma web diseñada para centralizar, analizar y poner a disposición del público información relacionada con la calidad del agua.
 El proyecto surge de la necesidad de contar con un espacio digital donde los usuarios puedan consultar datos actualizados, verificados y fáciles de entender sobre el estado del agua en su región, así como reportes de instituciones oficiales o publicaciones comunitarias.
La aplicación web cuenta con un sistema de registro de usuarios, lo que permite personalizar la experiencia y habilitar funcionalidades adicionales, como la publicación de reportes o comentarios sobre problemáticas relacionadas con el agua. Cada usuario registrado puede contribuir compartiendo información que pueda ser relevante para la comunidad, mientras que todos los datos oficiales provienen de fuentes reconocidas para garantizar su veracidad.
